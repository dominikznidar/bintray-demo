# JFrog Bintray demo

## About

This is quick test of seting up CI/CD flow for [Drone](https://drone.io) to publish artifacts to Jfrog Bintray

## How to use it

### Prepare the secrets

```
environment:
	JFROG_API_KEY: ##REPLACE_WITH_YOUR_API_KEY##

```

1. Clone the repo
2. Setup CI job on drone CI
3. Commit and push something
